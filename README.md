# Generador de estimados de inversión INVEX

Este proyecto contiene un formulario web estático que ayuda al equipo comercial a generar estimados rápidos para potenciales clientes. El flujo contempla los montos de implementación y las licencias mensuales para los diferentes perfiles de usuario.

## Uso

1. Abre el archivo `index.html` en tu navegador preferido.
2. Completa los datos generales del prospecto, la distribución de licencias y personaliza los textos del documento.
3. Haz clic en **Generar estimado** para visualizar la vista previa con los montos desglosados.
4. Ajusta los textos o cantidades según sea necesario. El bloque final indica claramente que el documento es un estimado.

Los precios de implementación utilizados son:

- 1 a 5 colaboradores: $7,000 USD.
- 6 a 15 colaboradores: $9,000 USD.
- Más de 20 colaboradores: contactar a un asesor para una propuesta personalizada.

Los costos mensuales por usuario son, con posibilidad de ajustarse desde el formulario:

- Administradores (todos los accesos): $350 USD.
- Ventas / Cajas: $250 USD.
- Inventarios: $300 USD.
- Otros módulos individuales: $250 USD.
- Usuarios combinados (hasta 3 módulos): $300 USD.

## Personalización

Las secciones de texto de la vista previa (encabezado, introducción, notas y aviso importante) pueden ajustarse directamente desde el formulario para adaptarse a cada cliente.

## Requisitos

No se requieren dependencias ni un servidor adicional: todo funciona directamente en el navegador.
